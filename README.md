FUTURE_DS_03 – Marketing Funnel & Conversion Performance Analysis - 

Overview- 
This project analyzes customer journey performance using October 2019 e-commerce behavioral data (~3M records). The objective was to evaluate funnel progression (View → Cart → Purchase), identify drop-off points, and assess category-level conversion efficiency.

Key Metrics - 
Total Visitors: 3,022,130
Total Purchasers: 347,118
Add-to-Cart Rate: 11.15%
Overall Conversion Rate: 11.49%

Key Insights - 
~11.49% of visitors converted to purchase.
Smartphone and appliance categories outperformed the platform average.
Mid-month spikes observed in daily purchase trends.

Tools Used - 
Power BI | DAX | Power Query | CSV Dataset

Dataset - 
E-commerce Behavior Dataset (October 2019)
